Editions

Microsoft offers different packages at various prices to match the specific situation of the developer or team. The following additions are available for Visual Studio 2015:

  • Visual Studio Community is a full-featured free package for individual, open source, or academic development. Functionality includes tools to integrate with Git and other revision control tools.
  • Visual Studio Professional is a package for professional development by an individual or small group. Licenses for this package start at approximately $45/month.
  • Visual Studio Enterprise is a package for professional development by teams of any size. Functionality includes tools to control development in an Agile environment. Licenses for this package start at approximately $250/month.

These editions allow for development teams of various sizes and budgetary price points develop great applications. This also allows engineers from different types of groups to collaborate across different editions on Visual Studio.

Microsoft also provides many extension programs which interact with any addition of Visual Studio to provide additional functionality. These extensions are optional and are often-times free to download based on your team composition and development need.

  • Team Explorer / Team Foundation Server is a client-server software package facilitating groups to share, collaborate, and track changes throughout the development process. The server-level can be hosted by a local tower or on the Microsoft Cloud (for a fee).
  • Visual Studio Code extends the development capabilities of Visual Studio to previously unsupported platforms/operating systems - Linux, OS X, or Windows. This extension also extends support for languages such as: R, CSS, SQL, Markdown, TypeScript, LESS, SASS, JSON, XML, Node.js, and Python.

As mentioned before, the strength of each edition and extension is independent of the development language used. Each edition supports numerous languages which are selectable by the developer at the time of installation.
