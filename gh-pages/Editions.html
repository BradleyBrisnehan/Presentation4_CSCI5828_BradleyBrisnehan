Editions

Microsoft offers different packages at various prices to match the specific situation of the developer or team. The following additions are available for Visual Studio 2015:

  • Visual Studio Community is a full-featured free package for individual, open source, or academic development. Functionality includes tools to integrate with Git and other revision control tools.
  • Visual Studio Professional is a package for professional development by an individual or small group. Licenses for this package start at approximately $45/month.
  • Visual Studio Enterprise is a package for professional developement by teams of any size. Functionality includes tools to control development in an Agile environment. Licenses for this package start at approximately $250/month.

These editions allow for development teams of various sizes and budgetary price points develop great applications.
