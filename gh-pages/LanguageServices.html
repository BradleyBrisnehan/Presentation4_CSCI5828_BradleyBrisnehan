Language Services

The Microsoft Visual Studio IDE supports the ability to develop and edit source code in the specific language of choice. The following is a list of some if the languages supported by Microsoft:

  • Microsoft Visual C++ offers compilation in either C or C++ mode. The C compiler follows the ISO C standard with support for portions of C99. The C++ compiler follows the ANSI C++ standard with support for portions of C++11.
  • Microsoft Visual C# is primarily targeted at application development within the .NET framework. The C# compiler supports version 5.0 of the language under the 2012 release.
  • Microsoft Visual Basic is primarily targeted at application development within the .NET framework.
  • Microsoft Visual Web Developer offers development of web applications, services, and sites using ASP.NET, C#.NET, or VB.NET.

As we will see when talking about editions, this list is by no means exhaustive. Microsoft offers extensions supporting more than 30 more languages, and the user always has the ability to create their own custom language service as part of a VSPackage. Rules for implementing a custom language service are outlined by MSDN (Microsoft Development Network) here.
