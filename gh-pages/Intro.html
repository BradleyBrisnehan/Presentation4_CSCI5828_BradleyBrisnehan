Introduction

In 1995 Microsoft merged four seperate offerings (Visual Basic 3, Visual C++, Visual FoxPro, and Visual SourceSafe) into a single product: Microsoft Visual Studio. Microsoft Visual Studio is an Integrated Development Environment (IDE) primarily used in the development of Windows applications and web services.

Visual Studio supports any number of languages using specific VSPackages referred to as Language Services. The following language services are supported by Microsoft for C, C++, VB.NET, C#, and F#. In addition, Visual Studio supports plug-ins at every level creating support for languages such as Python and Ruby as well as hardware specific functionality (Atmel Studio 7 Powered by Visual Studio).

Microsoft Visual Studio includes a code editor, debugger, and GUI Designer.

  • The code editor (like many others) supports syntax highlighting of variables, functions, methods, and loops. It also supports auto-completion of code using a technology referred to as IntelliSense.
  • The debugger works with managed as well as native code. It can monitor and debug a non-deployed process as well as provide memory dumps from deployed applications which can be loaded into the debugger at a later time.
  • The GUI designer allows easy manipulation/layout of objects within a window. Objects can be moved with a standard drag-and-drop manipulation. Windows Form Designer and Windows Presentation Foundation are two of the primary visual design tools used within Visual Studio.

The Just-In-Time debugger automatically launches Visual Studio when an unhandled exception is encountered during application execution. If undesireable in the final deployment, this functionality can be turned off.

Microsoft Visual Studio is a powerful tool and an important part of the software engineer's repertoire.
