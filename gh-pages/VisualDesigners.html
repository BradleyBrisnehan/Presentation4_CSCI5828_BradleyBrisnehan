Visual Designers

The Visual Studio graphic designer allows the creation of applications using drag-and-drop elements from the toolbox. All of these designers automatically generate source code behind the application. While this code can be manually edited, the graphical designers offer and abstraction to speed development.

  • Windows Forms Designer allows the design of application graphics with primitive Windows graphics. These objects generate code in C#.NET or VB.NET.
  • Windows Presentation Foundation (WPF) Designer allows the design of application graphics (including images, text, graphics, documents, and more) with more refined graphics. These objects are built on top of DirectX and generate code in XAML, an XML-based language. The appearance of the object is defined in XAML, and the functionality is defined in C# or VB.NET. This functionality was introduced under Visual Studio 2008.
  • Web Designer/Development allows the design of websites by dropping widgets. This supports development of applications in ASP.NET with support of HTML, CSS, and JavaScript.
  • Class Designer allows the development of classes using the (Unified Modeling Language) UML. These objects generate code in C#.NET or VB.NET.
  • Data Designer allows the definition of SQL database schemas, primary/foreign keys, and SQL queries for integration with the application.

With this set of visual designers, development of the application is simple and rich graphical interfaces are possible. Select the tools specific to your application and begin developing by dragging objects into the window.
